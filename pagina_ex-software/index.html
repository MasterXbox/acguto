Execução de um Software

Como funciona a abertura de um programa dentro de um computador

Após clicar em um dos programas da área de trabalho, a seguir temos a explicação de como funciona a execução de um software:



Após o usuário executar um comando no computador através de um dispositivo de entrada, o sistema operacional responde ao comando e começa a carregar o programa para a memória RAM, que tem a função de comandar a primeira instrução do programa que seja executada pelo processador.

Um software é nada mais do que uma série de instruções que são feitas através de uma linguagem de computação, para serem lidas e executadas pelo computador que após algumas instruções retorna os dados para que o sistema operacional, através de dispositivos de saída, mostre os dados para o usuário.
